# Hearthstone Assets

## Required Assets

### App Icons
- `icon.png` - 1024x1024 app icon
- `adaptive-icon.png` - 1024x1024 Android adaptive icon
- `favicon.png` - 48x48 web favicon

### Splash Screen
- `splash.png` - 1284x2778 splash screen (iPhone 14 Pro Max)

### Tab Bar Icons
All icons should be 24x24 with 2x and 3x variants:
- `tab-tonight.png` - Main meal recommendation (flame/plate icon)
- `tab-dashboard.png` - Stats dashboard (chart icon)
- `tab-inventory.png` - Pantry/fridge (refrigerator icon)
- `tab-prep.png` - Sunday prep (calendar/knife icon)
- `tab-quest.png` - Gamification (trophy/star icon)

### Food Category Icons
32x32 icons for inventory categories:
- `cat-protein.png`
- `cat-dairy.png`
- `cat-produce.png`
- `cat-grains.png`
- `cat-frozen.png`
- `cat-pantry.png`

### Misc
- `placeholder-meal.png` - Placeholder for meal images
- `empty-state.png` - Empty state illustrations
- `onboarding-*.png` - Onboarding flow images

## Brand Guidelines

### Colors
- Primary: Hearth Orange `#E85D04`
- Secondary: Sage Green `#52796F`
- Background: Cream `#FDF6E3`
- Text: Charcoal `#2D3436`

### Style
- Warm, inviting aesthetic
- Hand-drawn feel for illustrations
- Food photography should be appetizing and realistic
- Avoid overly polished/stock photo look

## Asset Generation

Icons can be generated using:
- [Expo Icon Builder](https://buildicon.netlify.app/)
- [App Icon Generator](https://appicon.co/)

For illustrations, consider:
- [unDraw](https://undraw.co/)
- Custom illustrations matching the warm, homey brand
