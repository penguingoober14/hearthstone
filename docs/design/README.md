## Design System

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Hearth Orange | `#E85D04` | Primary CTA, brand accent |
| Charcoal | `#2D3436` | Text, headers |
| Cream | `#FDF6E3` | Background |
| Sage Green | `#52796F` | Secondary, success states |
| Soft Red | `#D62828` | Warnings, expiry alerts |

### Typography
- **Headers**: System font, bold, 28px
- **Body**: System font, regular, 16px
- **Captions**: System font, regular, 14px

### Spacing
- Base unit: 4px
- Common spacing: 8, 12, 16, 20, 24px

### Border Radius
- Cards: 16px
- Buttons: 12px
- Inputs: 8px
